TOEFL Ten Thousand Words (1)

Posted on 2018-01-05 |

3.1

abjure v 宣布放弃 放弃权力

abandon

  • ab => away jure => to swear
  • adjure (ad => to) v 恳求
  • conjure v: 1, 恳求（一起发誓） 2, 变魔术（共同的誓言，不揭穿）
  • perjure （per => away, through）v 做伪证（背离誓言） -> n perjury

antonym

  • espouse (e => 加强) 支持
  • spouse 夫妻，配偶
  • spawn 卵

syn

  • forgo: (for => forward 先走一步) 放弃（人间的快乐 世俗的快乐）
  • forsake: （for the sake of, 为了原因而放弃） 放弃
  • forswear: (宣誓)放弃
  • renounce: (retire + announce) 放弃（权力，信仰）
  • relinquish: (re => back)(link ) (回到以前)（不情愿地）放弃

abominate v 痛恨

ab => away, omin => omen(凶兆, Amen) ominous 不吉祥的

  • abhor: (hor -> horrible) 厌恶，痛恨 (abhorrent 令人厌恶的)
  • detest: (de => off, deforest, dehydrate) （远离考试）痛恨
  • loathe: 痛恨 (-> adj loath, oath 痛恨誓言, loss)

abstain v 禁绝, 放弃

abstain from
abstinence 禁欲
abs后为t或c时，前缀为abs，而不是ab
abs => away (abstract, abscond[（一起离开）潜逃], absolve)
tain => to keep (maintain)

  • detain:(de => down 留下来) vt 扣留
  • retain: (re => back 保持回来) vt 保持（意见，想法）
  • pertain (+ to): (per => through) 1 属于 2 关于 (a. pertinent)
  • distain: (不保持) 贬低，贬损
  • disdain: 轻视 瞧不起

acerbic a 酸的， 尖刻的

ac/acerb/acr => 1 sour, 2 sharp
acid, ace

  • exacerbate: (ex => to 加强)(加强) (矛盾)变尖锐
  • acrid: 尖刻的， 辛辣的（嘲讽）
  • acrimonious: 同上
  • acme: [me -> 顶峰 (acne 你-> 粉刺)] 顶峰，顶点
  • acumen: [男人很尖] 精明，敏锐

3.2

affluence n 富裕，富足

  • [flu => flow 流动 流淌 很富足]
  • superfluous: 过剩的 [super -> 超级, on, more]
  • effluvia:
  • fluctuate
  • flux
  • influx

3.3

aloof adv/adj. 冷淡的，孤零的

a + 动/形容 ==> adj/adv (eg. aloud, alive)

loof -> leave (离我远点, 冷淡)

  • aground: < a/ad > 搁浅
  • afoot: < a/ad > 徒步的, 计划在进行中的
  • alight: < a > burning 燃烧着的，着火的
  • akin: [kin: relatives 亲戚] < a/ad > 同族的，同源的

syn

apathy: [a -> not, path => feeling eg. passion]

sympathy

antipathy: 反感

frigid:[fridge 冰箱] 冷淡的

offish: off+ish 远离的, 冷淡的

altuism n. 利他主义

  • [all + true] ism (所有的一切都非常真诚 利他)
  • egoism
  • egocentric: 以自我为中心
  • altr/alter => other (来自其他的变形) alternative
  • alternate v 交替 轮流
  • altercate [c 吵架]v 争吵 口角
  • adulterate

amalgam n 混合物 汞合金

  • amalgamate 1. mix 2. combine
  • coalesce [esce 逐渐…样的 esce v escent a escence n]联合 合并
  • adolescence [escence 逐渐成熟的] 青春期 少年

ambush v 潜伏 伏兵

  • [bush 藏在草丛里]
  • lush a 繁茂的 茂盛的 [lux 奢华的]
  • blush v 脸红 [不拉屎憋得满脸通红]
  • plush a 豪华的 奢华好 [lux 漂亮就是豪华]
  • gush v 涌出 喷出 [gu 鼓出 -> 涌出]
  • hush n/v 安静 肃静 [hush-hush 呼吸就要安静]
