abjure v 宣布放弃 放弃权力

abandon

  • ab => away jure => to swear
  • adjure (ad => to) v 恳求
  • conjure v: 1, 恳求（一起发誓） 2, 变魔术（共同的誓言，不揭穿）
  • perjure （per => away, through）v 做伪证（背离誓言） -> n perjury

antonym

  • espouse (e => 加强) 支持
  • spouse 夫妻，配偶
  • spawn 卵

syn

  • forgo: (for => forward 先走一步) 放弃（人间的快乐 世俗的快乐）
  • forsake: （for the sake of, 为了原因而放弃） 放弃
  • forswear: (宣誓)放弃
  • renounce: (retire + announce) 放弃（权力，信仰）
  • relinquish: (re => back)(link ) (回到以前)（不情愿地）放弃

abominate v 痛恨

ab => away, omin => omen(凶兆, Amen) ominous 不吉祥的

  • abhor: (hor -> horrible) 厌恶，痛恨 (abhorrent 令人厌恶的)
  • detest: (de => off, deforest, dehydrate) （远离考试）痛恨
  • loathe: 痛恨 (-> adj loath, oath 痛恨誓言, loss)

Geometry

Points, Vectors and Normals

Points and Vectors

Here, a point is a position in a three-dimensional space. A vector, on the other hand, usually means a direction (and some corresponding magnitude, or size) in three-dimensional space.

Normals

A normal is the technical term used in Computer Graphics (and Geometry) to describe the orientation of a surface of a geometric object at a point on that surface. Technically, the surface normal to a surface at point PP, can be seen as the vector perpendicular to a plane tangent to the surface at P.

Coordinate Systems

Cartesian coordinate system

For example, consider drawing a number of points on a piece of paper. This piece of paper occupies a two dimensional space, i.e. a plane. We can again draw two axes: one for each dimension. If we use the same x- and y-axes to measure each point drawn on that paper, these two axes are said to define a coordinate system. If these two rulers are perpendicular to each other, they define what we call a Cartesian coordinate system.

Euclidean space

The 3D coordinate system is a simple extension of the 2D case. We will just be adding a third axis orthogonal to both the x- and y-axis called the z-axis (representative of depth). In Geometry, this 3D coordinate system defines what is more formally known as Euclidean space.
