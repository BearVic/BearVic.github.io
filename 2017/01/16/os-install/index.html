电脑安装系统的几种方法总结

Posted on 2017-01-16 |

前言

因为旧的win本系统坏掉了，这两天就捣鼓着并学习了系统的安装方法，主要分为系统下安装新系统、无系统时候安装系统以及安装双系统。在这里简单的总结一下:)

系统下安装新系统

这个我是蛮写一下的，因为太简单了。下载个IOS系统映像文件直接安装就好了。

无系统时候安装系统

没有系统的时候就很蛋疼了，我是使用了个U盘当做系统启动盘安装的。估摸着分为一下几步吧（是弄完靠着回忆总结的，错误的地方见谅）：

  • 首先你需要有一台可以正常运行的电脑，下载安装将普通U盘（>8GB）制作成系统启动盘的工具软件，例如：ultraISO(软碟通 有试用版)，大白菜等。我用的是大白菜，ISO模式，集成的工具挺好用。然后还要下载你想安装的系统的映像ISO文件。然后通过大白菜将ISO文件写入U盘成系统盘。具体步骤就不详述，很简单。
  • 接着，很重要的一步（可能只是我的笔记本需要这个步骤），就是笔记本重启的时候F2进入那个什么页面，将F12进入boot置为可用enable。（我一开始以为是系统无法识别启动盘，弄了好久，最后才发现要先F12进入boot模式，然后才能进入USB启动）
  • 最后就是如上所述的，重启开机画面F12（F12是因系统而异）进入boot模式选择U盘启动，进入系统安装画面，然后就开始安装吧。

2017-07-29 更新

  • 用大白菜将U盘转换成系统启动盘
  • 将要安装的系统拷贝到转换成启动盘的U盘中
  • 重启电脑进入启动盘，在pe中安装系统
  • 注意：第二步中若系统文件>4GB，则需要在转换启动盘时设置成NFTS格式。

安装双系统

  • 同样要先制作一个系统启动盘，方法参加上一个条目。
  • 要在Windows下新划出一个大于20G的硬盘空间（本人划分了60G，如果说只是为了玩玩，那么20G足够了）在win7系统下->计算机->右键->管理->磁盘管理，选择你要安装的磁盘，压缩卷分配系统大小。
  • 然后就开始安装系统，如上个条目中介绍的，重启F12安装系统。具体安装过程没啥好说的，有几点需要注意：
    • 在安装类型界然后选择最后一项“其他选项”，以为这样可以自己手动分区，点击继续。
    • 现在我们看到的是硬盘的分区情况，找到前边有“空闲”二字，我们要做到就是，把空闲的空间给ubuntu划分分区。
    • 点击“空闲”的分区，选择下边的“添加”，弹出窗口如下，上边填写分区空间大小，下边填写要挂载的分区，然后确定。我的分配参考如下：
  • 分区设置好后，查看/boot分区的编号，然后在下边的“安装启动引导区的设备”下拉框中选择/boot分区的编号，点击安装。一直到你安装成功，当然中途需要你设置一个用户名和密码的（这个就不说了吧）
  • 最后重启，进入win7系统，下载安装easyBCD用于在重启时添加系统引导。
打开easyBCD，选择add new entry, 选择Linux/BSD,  
name这一行随便填写，只是系统名词，写ubuntu吧,  
Device这一行选择刚刚我们创建的200MB的那个”/boot“分区,  
前边有linux标记的。（其他的不要动）

然后重启系统，就会有个选择系统的画面。我的笔记本进入Ubuntu系统，特别慢，不知道是硬件的原因还是啥，但是能用不错了。
