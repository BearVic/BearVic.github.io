沙盒环境

virtualenv用于创建独立的Python环境，多个Python相互独立，互不影响，它能够：

  1. 在没有权限的情况下安装新套件
  2. 不同应用可以使用不同的套件版本
  3. 套件升级不影响其他应用

安装方法（进入安装文件夹）: 

pip install virtualenv

激活方法（进入安装文件夹）:

. /bin/activate

离开：

deactivate
